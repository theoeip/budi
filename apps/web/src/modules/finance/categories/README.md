# Categories

> Transaction category management.

## Purpose

Manages categories used to classify transactions:

- Creating and editing categories
- Income vs expense categories
- Category hierarchy (parent/child)
- Color coding for visual identification

## Components (Future)

- `CategoryList` — Category grid/list view
- `CategoryForm` — Add/edit category form
- `CategoryTree` — Hierarchical category view
