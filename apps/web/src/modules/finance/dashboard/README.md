# Finance Dashboard

> Financial overview showing key metrics and KPIs.

## Purpose

Provides a high-level view of the school's financial health including:

- Current balance overview
- Monthly income vs expense comparison
- Recent transactions
- Pending payments
- Category breakdown charts
