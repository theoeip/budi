# Reports

> Financial reports and exports.

## Purpose

Generates financial reports for the school:

- Income statement
- Expense breakdown
- Category summary
- Period comparison
- Export to PDF/Excel

## Components (Future)

- `ReportBuilder` — Report configuration form
- `ReportViewer` — Report display with charts
- `ReportExport` — Export options (PDF, Excel, CSV)
