# Finance Settings

> Module configuration and preferences.

## Purpose

Configures finance module behavior:

- Fiscal year settings
- Default currency
- Tax configuration (future)
- Automatic receipt generation
- Budget settings

## Components (Future)

- `FinanceSettingsForm` — Module configuration form
- `FiscalYearSettings` — Fiscal year configuration
- `CurrencySettings` — Currency preferences
