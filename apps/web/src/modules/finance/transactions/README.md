# Transactions

> Income and expense transaction management.

## Purpose

Manages financial transactions including:

- Recording income and expense entries
- Categorizing transactions
- Searching and filtering
- Bulk operations

## Components (Future)

- `TransactionList` — Paginated transaction table
- `TransactionForm` — Add/edit transaction modal
- `TransactionFilters` — Advanced filter bar
- `TransactionImport` — CSV import (future)
